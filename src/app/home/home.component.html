Deferrable Views

  • used to defer the loading of dependencies within a template
  • The dependencies include components, directives, and pipes, and any associated CSS 

@defer {{ '{' }}
    <large-component />
{{ '}' }}

Triggers

  • on idle
  • on viewport
  • on interaction
  • on hover
  • on immediate
  • on timer

Blocks

  • @defer
  • @placeholder
  • @loading
  • @error

Limitations

  • The dependencies must be standalone
  • Must not be directly referenced from the same file, outside of @defer blocks
